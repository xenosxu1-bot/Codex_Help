# Contributing

Contributions are welcome.

Please keep changes aligned with the official Codex documentation:

1. Verify product facts against https://developers.openai.com/codex.
2. Avoid adding undocumented claims about pricing, availability, limits, or enterprise policy.
3. Prefer concise, practical examples that help readers start using Codex safely.
4. Keep Chinese and English docs factually consistent.
5. Update links and screenshots only from official or clearly attributed sources.

For factual updates, include the official documentation page that supports the change.
